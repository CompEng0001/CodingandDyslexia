# Coding and Dyslexia
This repo is collate coding support for those who have neurodiversity (dyslexia, ADHD etc), 

## Notes
 - [OpenDyslexic](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC5629233/) has shown that it may not work for all dyslexics.
